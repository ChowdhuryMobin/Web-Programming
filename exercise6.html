Cascasing Style Sheets

  • I would suggest the school to use Cascading Style Sheets which can be used to control the Webpage.
  • Cascading Style sheets can be used to separate, organize and style the various departments of the school. It will make the website consistant and easy to use and find items.
  • If the school wants to change their logo, update pictures and male changes it can be done easily by this
  • The style sheet provide quick and easy acess which can be used to change website design in future.
  • If we let every teacher write their own style it will make the website inconsistent. There will be a posibilty to override styles, loss of images and change in main layout. It will be difficult to improve the website and keep it consistent in future.

HTML Tags and Classes for Website

  1. h1,h2,h3,h4,h5,h6 tags for Headings
  2. table tag for tables
  3. body tag
  4. header
  5. link for stylesheet
  6. p tags for paragraphs
  7. section tags to separate sections of website for easy access
  8. article tag for article
  9. class tags for targeting different sections of the page
  10. a tags to links external websites
